About

Semantic web forms is an easy and intuitive way to generate semantic web instances without having to write any code. The builder uses OWL files to read pre-existing ontologies and class definitions.

Given a set of Ontologies, the Form builder generates a Tree wherein you can select the classes you need in your Ontology. Once selected, we will generate intuitive forms for you to create instances of these classes. Try our builder now!
